# 🚗 Car Parking Management System

A modern full-stack **Car Parking Management System** developed using **Node.js, Express.js, SQLite3, HTML, CSS, and JavaScript**. The application allows users to park vehicles, manage parking records, track slot availability, calculate parking fees, and view reports through a responsive dashboard.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/car-parking-system.git
```

Go to project directory

```bash
cd car-parking-system
```

Install dependencies

```bash

```

Start the server

```bash
npm start
```

Open

```
http://localhost:5000
```

# 📡 REST APIs

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /cars | Fetch parked vehicles |
| POST | /park | Park a vehicle |
| POST | /exit | Exit a vehicle |
| POST | /update | Update vehicle details |
| GET | /slots | View parking slot status |
| GET | /report | Generate parking report |
